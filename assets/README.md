# Assets Directory

This folder stores static files used by audio and image commands.

Expected contents include:
- intro clips
- bait libraries (audio/image)
- ICC profile files
- helper segments (prefix/cinnamon audio)

Operational guidance:
- Keep only files that are safe to distribute publicly.
- Avoid committing copyrighted or private media unless you have rights.
- Use consistent naming for easier moderation and command debugging.
